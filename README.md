EcoCredit ‚Äì Plastic Recycling Reward Management System (DBMS Project)

EcoCredit is a full-stack web application designed to promote plastic recycling by rewarding users with eco-credits based on the type and weight of plastic recycled. This project was developed as part of my Database Management Systems (DBMS) coursework, with a strong focus on SQL database design, backend integration, and role-based access control.

The system allows users to submit plastic recycling data, track earned credits, and redeem eco-friendly rewards, while administrators can verify submissions, manage approvals, and monitor overall recycling impact.
